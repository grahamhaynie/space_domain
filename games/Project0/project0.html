Project 0 - Initial Animation

Type: Animation

Concepts: Basic animation

Description:

This being the first project, the goal was to learn basic JavaScript using the Processing library. To accomplish this, a logo (my initials) with animation was created using basic shapes. The small white spheres are generated with coordinates outside of the animation window, and are projected inwards. Once they collide with invisible rectangles, they are locked in place. If a ball reaches the middle of the window without being locked in, it is regenerated outside the window. Fun fact: Each instance of the animation produces different results, as the ball's initial positions are randomly generated.
