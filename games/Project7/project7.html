Project 7 - Sheep Maze

Type: Game

Concepts: Graph Traversal

Description:

The purpose of this project was to create a game utilizing A* search. The player moves around a pre-set 20x20 maze, attempting to outsmart NPCs that are constantly moving and can not get stuck. Every time the player moves to a new position, each adversary re-calculates the optimal path to the player. For each game instance, the player and sheep positions are randomized within a set distance range of eachother.

Gameplay:

Read the instructions and click the game screen to begin. Move the black dog character with the arrow keys around the maze. Try to get the sheep to chase you, taking care not to get caught, into the pen in the center. You can not travel through the pen, so be creative. If the sheep catch you, you lose. If you can corral both sheep, you win.
