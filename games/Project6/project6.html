Project 6 - Alien Zoo

Type: Animation

Concepts: Particle Systems, Subdivision, Bezier Curves, Perlin Noise

Description:

The purpose of this project was to learn about techniques used to animate video games in ways that make them appear realistic. This was done in the context of an alien zoo, where fish move from left to right, bigger fish eating smaller fish. Flying creatures also roam the sky above the fish tank, where fountains spew rainbow liquid. The fountain, as well as the blood pools created by fish eating each other, are created using particle systems. Subdivision and Bezier curves are used to draw the fish and flying creatures. Perlin noise is used to draw the fish tank water, which changes color gradually. As the water changes color, the fish change color as well. Bubbles also float from the bottom of the fish tank upwards, and appear in a small group when fish are eaten.
