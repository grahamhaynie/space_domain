Project 1 - Hyperspace

Genre: Shoot-em-up Animation

Concepts: Game loop, game architecture, action + interaction

Description:

A continuation of the previous project, the animation from Project 0 is used as the starting screen. When the player clicks the letter containing the green trianglle, they are presented instructions and can start the game. The focus of this was creating a game based on user input - clicking to control the menu and the left/right arrow keys to control the game's player.
