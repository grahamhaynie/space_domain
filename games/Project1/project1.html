Project 1 - Hyperspace

Type: Game

Concepts: Game loop, game architecture, action + interaction

Description:

A continuation of the previous project, the animation from Project 0 is used as the starting screen. The focus of this was creating a game based on user input - clicking to control the menu and the left/right arrow keys to control the game's player. The player controls a space ship in an endless asteroid field, attempting to dodge asteroids and collect fuel. Sadly I never got around to the art part of this game, so it looks a little bare bones.

Gameplay:

Click the 'G' once the green arrow appears to begin the game. Once you've read the instructions, click the game's window anywhere to start the game. To play, move the red player left/right with the left/right arrow keys. Try to dodge the grey asteroids and collect the green fuel spheres. When you run into an asteroid or run out of fuel, the game is over. You can click again in the game's screen to return to the starting menu.
