Project 2 - Robot Factory

Type: Game

Concepts: Background tile-map, custom characters, collision, keyboard control

Description:

The purpose of this project was to create a tile-map game in which the player controls a character with a keyboard. The project also taught about the creation of custom collisions (which I will admit are not great in this project). The characters are drawn using built-in shapes. The tile-map is loaded in from a long string array, and is static (the same every time). Everything is generated from the tile-map.

Gameplay:

Click the starting screen to begin the game. Move the black robot around the map with the arrow keys. Try to collect all of the blue discs, while avoiding running into any purple robots. Once you collect all of the discs, you win. If a robot catches you, you lose and must try again.
